#simple to-do list application
A basic To-Do List web app built using HTML, CSS, JavaScript, Node.js(Express), and MySQL.

#Features
>Add new tasks
>View all tasks
>Delete tasks
>Data stored in MySQL database
(update feature to be added)

#Tech Stack
Frontend: HTML, CSS(Bootstrap), Javascript
Backend: Node.js, Express
Database: MySQL

>Edit/Update functionality
>Mark as completed
>Better UI
>SignIn Login funtionality for privacy
